Installation
############

Install with `pip3`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install beem

or the with `pip`:

::

    $ pip install -U beem

Manual installation:

::

    $ git clone https://github.com/holgern/beem/
    $ cd beem
    $ python setup.py build
    $ python setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
